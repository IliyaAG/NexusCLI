# Quick Setup:
## Build From Source Code
**1)** clone the project:
```bash
git clone https://github.com/IliyaAG/NexusCLI
```
**2)** go to project directory:
```bash
cd NexusCLI
```
**3)** excutble setup.sh:
```bash
chmod +x setup.sh
```
**4)** run setup.sh and select your main shell:
```bash
./setup.sh
```
